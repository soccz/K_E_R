# 일간 메모 (Daily Note) Spec

## 0. 목적

**외부 노출용 짧은 투자 리서치 노트**. 사용자가 퀀트·애널리스트 취업 포트폴리오로 활용하기 위한 공개 관찰 기록.

내부 트랙(사업보고서 풀 진단)이 *깊이*를 담당한다면, 외부 트랙(일간 메모)은 *시점·일관성·관찰자 시각*을 담당한다. 매주 1편의 깊은 보고서 + 주 2~3편의 짧은 관찰이 누적되면 1년 후 *연구자 톤의 시계열 관찰 일지*가 된다.

## 1. 핵심 원칙

### 1.1 페르소나 100% 유지

`_persona.md`의 모든 규칙은 일간 메모에도 적용된다:
- 출처 엄격주의 (§3) — 모든 사실에 1순위 출처
- 비관 편향이 기본 (§2)
- 매수·매도 권고·목표가 금지 (§8)
- 양다리 결론 금지 (§8)
- 외인 vs 국내 자금 디커플링 ★★★ 신호 (§11.1)

### 1.2 학술/연구 톤 추가

페르소나 톤 위에 *연구자 어휘*를 덧입힌다:

| 셀사이드 톤 (금지) | 페르소나 톤 (내부 보고서) | 학술 톤 (일간 메모) |
|---|---|---|
| "Buy 200만원" | "지금 가격에 통째로 살 만한가" | "관찰된 신호 — 다음 검증 포인트" |
| "상승 가능성 70%" | "상승 가정의 가격" | "X 가설을 검증할 수 있는 첫 단서" |
| "매수 추천" | "사업 자체는 매력적" | "본 노트는 매수·매도 권고가 아님" |
| "주가 모멘텀 긍정" | "국내 환호의 역지표" | "가격-외인 동조성 관찰됨, 디커플링은 미발현" |

학술 어휘 예시:
- *관찰된다*, *해석된다*, *해석 가능*
- *가설*, *검증 포인트*, *시그널*, *단서*
- *재배분*, *디커플링*, *동조성*, *발현*
- *3~4분기 추적이 필요하다*, *현재 시점에서는 결정적이지 않다*

금지 어휘 (페르소나 §8 + 연구 톤 위반):
- "추천한다", "사야 한다", "팔아야 한다"
- "X% 오를 것이다" — 정량 예측 금지
- "단기 모멘텀 + Buy" — 셀사이드 흉내 금지
- 1인칭 감정 ("좋다", "나쁘다", "기대된다")

### 1.3 1인칭·감정·확신 회피

연구자는 *데이터를 관찰*하고 *가설을 제시*한다. 결론을 *확신*하지 않는다.

```
✗ "SK하이닉스가 매력적이다"
✓ "SK하이닉스의 시총/FCF 47배는 사이클 정점 가격으로 해석된다"

✗ "외인이 빠지고 있어서 위험하다"
✓ "3일 누적 외인 -1,200억은 추세적 매도의 첫 단서로 분류된다 — 5일 누적 조건이
  충족되면 페르소나 ★★★ 디커플링 신호의 정량 발현으로 본다"
```

## 2. 출력 구조 (1면)

```markdown
# YYYY-MM-DD 일간 메모            [트리거 헤드라인 1줄]

## 매크로 (YYYY-MM-DD 종가)
KOSPI    [값] [등락률] [60일 스파크라인]
USD/KRW  [값] [등락률] [60일 스파크라인]
WTI ($)  [값] [등락률] [60일 스파크라인]

## 관찰 (3~4줄)
[오늘의 트리거 데이터 + 페르소나 ★★★ 신호 매핑 + 학술적 해석]

## 종목 카드 (1~3개)
### 종목명 (티커)                   [등락률]
[60일 종가 SVG 스파크라인]
[수급·거래대금·외인 데이터 1줄]
[학술 톤 멘트 2~3줄: 관찰 → 해석 → 다음 검증 포인트]

## 섹터 관찰
[반도체 ↑] [2차전지 ↓] [조선 ↑] [금융 →]  (한 줄 화살표)

---
> **트리거**: [임계치 충족 항목 명시]
> **목적**: 본 노트는 매수·매도 권고가 아닌 *공개 관찰 기록*. 가설·검증 중심.
> **이전 메모**: [최근 3편 링크]
```

분량 목표: **마크다운 350~500단어 / 렌더 후 1면**.

## 3. 트리거 (4종 임계치 — 종합)

매일 16:00 KST(장마감 +30분)에 4종 임계치를 검사. *어느 하나라도* 충족하면 LLM 호출 → 일간 메모 생성. 주 2~3회 발행 목표.

### 3.1 가격 변동 (워치리스트 24종목)
- 일별 등락률 ±5% 이상 → 트리거
- 데이터: KRX OHLCV (pykrx)

### 3.2 외인 수급 추세 (워치리스트 24종목)
- 3일 누적 외인 순매매 ±500억원 이상 → 트리거
- 데이터: KRX 투자자별 순매매 (KRX 계정 발급 후 v0.4. 현재는 placeholder)
- 임시 폴백: yfinance 거래량 변화 또는 DART majorstock 신규 공시

### 3.3 DART 주요 공시 (워치리스트 24종목)
- 잠정실적·M&A·정정공시·자사주 소각·주요사항보고 → 트리거
- 데이터: DART list.json (`pblntf_ty=A,B`)

### 3.4 외인-국내 디커플링 (★★★ 페르소나 신호)
- 주가 ±2% + 외인 매매 *반대 방향* (3일 누적 ±300억원) → 트리거
- 페르소나 §1.7 "국내 환호 역지표" 정확 발현
- 데이터: 위 3.1 + 3.2 결합

### 3.5 KOSPI200 큰 동향 (워치리스트 외)
- KOSPI200 종목 중 일별 ±7% (워치리스트보다 더 빡빡) → 트리거 + 종목 카드 1개
- 워치리스트 24 + 그날 특별히 큰 동향 종목 = 종목 풀

## 4. 종목 카드 자동 선정

트리거 통과 종목들을 점수화 후 상위 1~3개:
```
score = α * |등락률| + β * |외인 누적| + γ * 공시 임팩트 + δ * 디커플링 강도
α=20, β=0.5, γ=30, δ=50  (디커플링이 가장 무거움)
```

상위 3개를 종목 카드로. 1~2개만 임계치 통과한 날은 1~2개만 노출.

### 5.2 사이트 메인 메뉴
```
projects/k-e-r/
```

```
  - 실행: pipeline.daily_trigger 검사 → 트리거 시 daily_note_builder 호출
  - 후속: bin/publish_site.sh 자동 호출 → soccz.github.io push
```

## 7. 검증

일간 메모도 `_persona.md` + 본 spec을 따르므로:
- V1 (출처·추론·구조) 적용 — vague_source / 추론 마커 없는 추측 → fail
- V2 (XBRL 자릿수) — 일간 메모는 XBRL이 적어 적용 약함, 그러나 외부 데이터 키워드 skip
- V3 (산술) — 표가 있으면 적용
- V4 (cross-section) — 일간 메모는 단일 문서라 cross-section X. 그러나 *직전 30일 메모와의 일관성*을 별도 검증 가능 (V5 신규, 향후)
- 사이트 렌더 — 빌드 검증 통과해야 push
